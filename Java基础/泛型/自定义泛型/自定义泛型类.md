**自定义泛型类**

&nbsp;

**class 类名 \<T,R...\> { //表示可以有多个泛型**

**&nbsp; &nbsp; 成员**

**}**

**注意细节**

1. **使用泛型的数组，不能初始化，因为不能确定类型就不能确定数组开辟的空间**

**&nbsp;** Type parameter 'T' cannot be instantiated directly

2. **静态方法中不能使用类的泛型**

**&nbsp;** 因为静态是和类相关的，类加载时，对象还没有真正创建，即泛型还未确定

&nbsp; 故而如果静态犯法和静态属性使用了泛型，VM就无法完成初始化
