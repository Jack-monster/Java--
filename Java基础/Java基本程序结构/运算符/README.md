## 强制类型转换
如上，不能通过合法数值类型转换时，可以通过强制类型转换使数值得到转换。
```java
    double x = 9.997;
    int nx = (int)x;
```
强制类型转换通过截断二进制值实现，当转换是由浮点型转向整型时，会切断小数部分。
## 赋值中的强制类型转换
在赋值二元运算的简写形式“+=、-=。。。”中，如果运算符得到的数的类型与左侧数的类型不同，则会发生强制类型转换。

```java
    int x = 1;
    x += 3.5;
//相当于
    int x = 1;
    x = (int)(x + 3.5);//结果为4
```
